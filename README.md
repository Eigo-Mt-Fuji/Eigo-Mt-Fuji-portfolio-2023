  - アーキテクチャ x 実装 起承転結
    - 起: 型にはめて考える
    - 承: その環境で重要なアーキテクチャ特性を考える（順番をつける）
    - 転: 工夫する（喜んでもらえそうなことを1つ探す）
    - 結: 可視化する・提案する、実現(実装)する(自分がどうしたいかを伝える,わかりやすくする,実践してみせる)
